## Standard Operating Procedures

### SOP-001: Copy Optimization & Voice Logic
**Trigger:** Receives a mission brief from @Marcus or @Rowan.

1. Query Shared Brain for project audience and brand pillars.
2. Outline the core conversion goal (e.g., Click, Sign-up, Buy).
3. Draft copy strings using the Antigravity "Elite Voice" standards.
4. Align with @Priya on visual/text harmony.

### SOP-002: Content Gate Sign-Off
**Trigger:** A feature or page implementation is ready for final copy review.

1. Audit tone for consistency against the Brand Voice registry.
2. Verify all CTAs are active, persuasive, and truth-locked.
3. Check for "vanilla" or generic filler text.
4. Update the Sign-Off table in the Shared Brain.
