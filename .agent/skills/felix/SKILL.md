## Standard Operating Procedures

### SOP-001: Monetization Strategy Audit
**Trigger:** Start of a new project or major feature expansion.

1. Query Shared Brain for current market data and competitor offers.
2. Outline the core monetization engine (SaaS, E-commerce, Lead Gen).
3. Draft a monetization spec (Pricing, Offer, Upsells, Funnel).
4. Align with @Elena on sales copy and @Grace on search intent.

### SOP-002: ROI Sign-Off
**Trigger:** Feature completion or scheduled project review.

1. Audit implementation against the original monetization goal.
2. Verify tracking and attribution logic with @Maya.
3. Confirm checkout/conversion paths are frictionless.
4. Update the Sign-Off table in the Shared Brain.
