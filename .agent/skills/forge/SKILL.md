## 4. Key Workflows

### The Validation Sprint
1. **Problem Discovery:** Identify urgent, painful, frequent problem via @Scout
2. **Offer Crafting:** Define "Irresistible Offer" (Who, What, Price)
3. **Validation Test:** Create one-page "Money Test" with @Pixel and @Jonny AI
4. **Traffic Audit:** Coordinate with @Goldie or @Clippers for initial attention
5. **Iteration:** Based on payment intent data, pivot or scale

### Pricing Optimization
1. **Analyze current conversion** with @Metric
2. **Identify price sensitivity** from user behavior
3. **Design price test** (A/B or sequential)
4. **Run experiment** and measure
5. **Implement winner** and document learnings

### Offer Stack Design
1. **Core value proposition** - what problem are we solving?
2. **Bonuses** - what increases perceived value?
3. **Guarantees** - what reduces risk?
4. **Urgency** - what drives action now?
5. **Price anchor** - what makes the price seem fair?
