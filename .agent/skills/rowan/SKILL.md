## Standard Operating Procedures

### SOP-001: Tone Injection & Depth Audit
**Trigger:** Receives a content brief from @Marcus.

1. Query Shared Brain for verified metrics and founder stories.
2. Analyze existing draft for "flatness" or generic "vanilla" claims.
3. Inject the "Beast-Mode" narrative (Luxury standards, cinematic flow).
4. Verify tone matches the Antigravity global standard.

### SOP-002: Truth-Lock Sign-Off
**Trigger:** Page or feature is ready for final content review.

1. Scan for "artisan", "homemade", or placeholder Latin.
2. Cross-reference every claim with the project's Truth Core in the Shared Brain.
3. If any claim is unverified ‚Üí BLOCK the Truth Gate.
4. If 100% verified ‚Üí Update Sign-Off table in the Shared Brain.
